# JS-Restuarant-Page 
The goal of this project is to develop a restuarant page using DOM manipulation and NodeJS.

## Description
This is a project based on the activities in the JavaScript path of TheOdinProject Web Development Course.

This project is a modern take on a restuarant page from OdinProject for an example [Restuarant](https://web.archive.org/web/20221024060550/https://eckben.github.io/bearysBreakfastBar/) 

## Technologies and Languages Used:
1. HTML
2. CSS
3. JavaScript
4. NodeJS
5. NPM 

## What skills I have developed:
1. Application of NodeJS in Web Development
2. DOM Manipulation for creating HTML Elements
3. Styling on DOM created HTML Elements
